# AI for Healthcare WiDS'25

## Overview
This repository contains my learning and practical work in Computer Vision and Deep Learning, completed over three weeks. It covers image fundamentals, neural networks, medical imaging, and YOLO-based model training.

---

## Week 1: The Science Behind Images
**Concepts learned**
- Introduction to Computer Vision  
- Human Vision  
- Color Spaces (RGB, Grayscale)  
- Image Transforms  
- Image Coordinates  
- Image Resizing  

**Healthcare Reports**
- Studied different types of healthcare reports  
- Focused on medical imaging reports such as X-ray, CT, and MRI  

---

## Week 2: Neural Networks and CNNs
**Concepts learned**
- Basics of Neural Networks  
- How Neural Networks Learn  
- Backpropagation  
- Convolutional Neural Networks (CNNs)  

**Tasks Completed**
1. **Image Processing in Python**
   - Grayscale conversion  
   - Histogram visualization  
   - Image resizing  
   - Edge detection  
   - Noise addition and noise removal using Gaussian filter  

2. **Medical Imaging Dataset Exploration**
   - Explored two medical imaging datasets  
   - Identified image type, number of images, and classes  
   - Observed dataset imbalance and challenges  

---

## Week 3: CNNs, Darknet, and YOLO
**Concepts learned**
- Darknet framework  
- ImageNet fundamentals  
- CNN architectures: AlexNet, VGG, ResNet, EfficientNet  

**YOLO and Ultralytics**
- YOLO modes: Train, Val, Predict, Export, Track  
- YOLO tasks: Detect, Segment, Classify, Pose  

**Practical Work**
- Explored a skin disease dataset using Roboflow  
- Trained a YOLO-based classification model using GPU on Colab/Kaggle
